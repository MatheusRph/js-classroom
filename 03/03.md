# ⏱️ Routes & Controllers

## 📁 Pasta: `routes/`
> Define as URLs (caminhos).
- 📄 `userRoutes.js`

## 📁 Pasta: `controllers/`
> Define a lógica (o que fazer com os dados).
- 📄 `userController.js`

---

## 🧪 Exemplo de Divisão

**userController.js**
```javascript
exports.boasVindas = (req, res) => {
  const { nome } = req.params;
  res.json({ msg: `Usuário ${nome} acessado via Controller!` });
};
```

**userRoutes.js**
```javascript
const express = require("express");
const router = express.Router();
const userController = require("../controllers/userController");

router.get("/:nome", userController.boasVindas);

module.exports = router;
```
